# IdCapture - Document Types

This section lists the supported document types for the **IdCapture** component in the **NubariumSDK**.
Each document is identified using a simplified key format that combines the ISO-3166-1 alpha-3 country code and the document type.

| Country     | Document Type | Document Class | ISO3  |
| ----------- | ------------- | -------------- | ----- |
| Colombia    | `COL_IDCARD`  | `IdCard`       | `COL` |
| Ecuador     | `ECU_IDCARD`  | `IdCard`       | `ECU` |
| El Salvador | `SLV_IDCARD`  | `IdCard`       | `SLV` |
| Guatemala   | `GTM_IDCARD`  | `IdCard`       | `GTM` |
| Honduras    | `HND_IDCARD`  | `IdCard`       | `HND` |
| Nicaragua   | `NIC_IDCARD`  | `IdCard`       | `NIC` |
| Panama      | `PAN_IDCARD`  | `IdCard`       | `PAN` |
| Peru        | `PER_IDCARD`  | `IdCard`       | `PER` |
| Venezuela   | `VEN_IDCARD`  | `IdCard`       | `VEN` |

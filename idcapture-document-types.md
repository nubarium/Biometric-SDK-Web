This section lists the supported document types for the **IdCapture** component in the **NubariumSDK**.  
Each document is identified using a simplified key format that combines the ISO-3166-1 alpha-3 country code and the document type.

**Key format:** `<ISO3>_<DOCUMENT_TYPE>`  
**Example:** `MEX_IDCARD`

| Country | Key | Document Type | Document Class | ISO3 |
|---|---|---|---|---|
| Colombia | `COL_IDCARD` | `IDCARD` | `IdCard` | `COL` |
| Ecuador | `ECU_IDCARD` | `IDCARD` | `IdCard` | `ECU` |
| El Salvador | `SLV_IDCARD` | `IDCARD` | `IdCard` | `SLV` |
| Guatemala | `GTM_IDCARD` | `IDCARD` | `IdCard` | `GTM` |
| Honduras | `HND_IDCARD` | `IDCARD` | `IdCard` | `HND` |
| Nicaragua | `NIC_IDCARD` | `IDCARD` | `IdCard` | `NIC` |
| Panama | `PAN_IDCARD` | `IDCARD` | `IdCard` | `PAN` |
| Peru | `PER_IDCARD` | `IDCARD` | `IdCard` | `PER` |
| Venezuela | `VEN_IDCARD` | `IDCARD` | `IdCard` | `VEN` |
